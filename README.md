# TodoApp

A Flutter project for managing daily tasks.

This project is a starting point for a Flutter application that allows you to:

-Add, edit, and delete tasks

-Mark tasks as done or not done

-Save tasks automatically between sessions using SharedPreferences

-Toggle between Light and Dark themes
